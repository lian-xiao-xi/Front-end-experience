迈克尔 乔丹
历史上最伟大的篮球运动员。NBA篮球名人堂。乔丹是历史上最值钱的体育偶像之一，他是Nike，Gatorade，Hanes，McDonald's和MCI的主要代言人。他在1988年首次出现在Wheaties box的广告上并随即成为了他们的代言人。Nike为乔丹设计的系列签名鞋叫做“Air Jordan”。后来Nike把这条生产线从自己的厂家分出来，专门叫做“乔丹牌”。
